# World Clock Map (Flutter)

A simple Flutter application that allows users to click anywhere on an interactive OpenStreetMap and see the current local time for that position.

## Features
- Interactive OpenStreetMap using `flutter_map`.
- Offline timezone lookup from coordinates using `lat_lng_to_timezone`.
- Digital clock display in a responsive bottom sheet.
- Real-time clock updates.

## Technical Details
- Built with Flutter.
- Uses `timezone` package for local time conversion.
- Uses `intl` for date and time formatting.
